# 🏡 DecoraAI

### AI-Powered Interior Design Inspiration & Planning Platform

DecoraAI is a modern full-stack web application that helps users discover, manage, and showcase beautiful interior design ideas. Users can browse designs by category, search and filter interiors, view detailed design information, and contribute their own creations after authentication. The platform also includes a simple AI Design Assistant that provides interior design suggestions and inspiration based on user prompts. Built with a clean, responsive interface, DecoraAI delivers an engaging experience across all devices.

---

# ✨ Features

- 🔐 Secure user authentication with Better Auth
- 🏠 Browse beautiful interior design inspirations
- ➕ Add new interior design projects
- ✏️ Update and manage your own designs
- 🗑️ Delete interior designs
- 📖 Detailed design information page
- 🤖 AI Interior Design Assistant for design suggestions
- 🔍 Search interior designs by title or style
- 🗂️ Filter designs by category
- 💰 Sort designs by budget
- 📱 Fully responsive design for all devices
- 🎨 Clean and modern user interface
- ⚡ Fast loading and optimized performance
